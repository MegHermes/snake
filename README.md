# How to play:
- Download repo
- Open Terminal and navigate to folder (ie. cd ~/desktop/Java-Snake-Game-main)
- Type: javac -d bin snakey/src/snakey/*.java
- Type: java -classpath bin snakey.Game
- The game should run and open automatically
